# 프로젝트 1 — 나의 엔지니어링 포트폴리오

> **렛유인 AI + 바이브코딩 강의** | 4차시 수료 후 첫 번째 주말 프로젝트

---

## 개요

Gemini AI와 GitHub Pages를 사용해 **나만의 인터랙티브 엔지니어링 포트폴리오 웹페이지**를 만드는 프로젝트입니다.
코딩 지식 없이도 AI 프롬프트만으로 전문적인 웹사이트를 완성할 수 있습니다.

- **수행 기간**: 4차시 이후 주말 2일 (토~일)
- **사용 도구**: Gemini AI + GitHub Pages
- **결과물**: 인터넷에 공개되는 나만의 포트폴리오 URL

---

## 학습 목표

1. Gemini AI로 HTML/CSS/JS 코드를 생성하는 방법을 익힌다
2. GitHub Pages로 정적 웹사이트를 무료 배포한다
3. 자신의 관심 산업(반도체, 디스플레이, 2차전지, 바이오, 에너지)을 데이터로 표현한다
4. AI 도구를 엔지니어링 실무에 연결하는 감각을 기른다

---

## 주제: AI로 나를 소개하고 산업을 분석하는 웹페이지

### 필수 포함 내용
- **자기소개**: 이름(또는 닉네임), 전공, 관심 직무, 한 줄 슬로건
- **관심 기술 TOP 3**: 내가 배우고 싶은 엔지니어링 기술 카드
- **관심 산업 분석**: 아래 중 하나 선택, 데이터/차트 포함

### 관심 산업 선택지 (하나 이상 선택)

| 산업 | 분석 주제 예시 |
|------|--------------|
| 반도체 | 웨이퍼 수율 트렌드, 공정 노드 비교 |
| 디스플레이 | OLED vs LCD 시장/성능 비교 |
| 2차전지 | 셀 용량·에너지밀도 비교, 양극재 종류 |
| 바이오 | 신약 파이프라인 단계, 임상 성공률 |
| 에너지 | 발전원별 효율·탄소 배출 비교 |

## 난이도별 수행 가이드

### 하 (기본) — Gemini 프롬프트 복붙 → 배포
**목표**: AI가 만들어 준 코드를 수정 없이 배포해도 OK. 과정을 경험하는 것이 핵심.

**단계별 가이드**:
1. [Gemini](https://gemini.google.com)에 접속
2. 아래 프롬프트를 복사해서 붙여넣기:

```
반도체 공정 엔지니어를 꿈꾸는 공대생의 포트폴리오 웹페이지를 만들어줘.
- 다크모드 디자인
- 자기소개 섹션 (이름: [본인 이름], 전공: [본인 전공])
- 관심 기술 TOP 3 카드 (반도체 공정, 포토리소그래피, 수율 분석)
- 반도체 주요 공정 단계 비교 테이블
- 모든 내용은 한국어로, 하나의 HTML 파일로 완성
```

3. 생성된 HTML 코드를 복사 → `index.html` 파일로 저장
4. GitHub 계정 생성 (없으면)
5. 새 레포지토리 생성 → `index.html` 업로드
6. Settings → Pages → Branch: main → Save
7. 배포 URL 확인 및 제출

**평가 기준**: 배포 완료 여부, 자기소개 포함 여부

---

### 중 (응용) — 실제 데이터 + 인터랙티브 차트
**목표**: 관심 산업 실제 데이터를 포함하고, 직접 작성한 프롬프트로 차트를 구현.

**요구사항**:
- 인터랙티브 차트 1개 이상 (Chart.js 사용)
- 실제 또는 실제처럼 보이는 산업 데이터
- 탭 또는 섹션 전환 UI
- 프롬프트를 직접 작성 (템플릿 없음)

**힌트**:
- "Chart.js CDN을 사용해서..." 라고 Gemini에게 요청하면 외부 라이브러리 설치 없이 차트 생성 가능
- 여러 번 수정 요청해도 됨: "이 차트를 막대 그래프로 바꿔줘", "색상을 더 진하게"

**평가 기준**: 차트 구현 + 데이터 출처 명시 + 프롬프트 기록 제출

---

### 상 (심화) — 완성형 포트폴리오
**목표**: 취업 포트폴리오로 실제 사용 가능한 수준의 웹페이지 완성.

**요구사항**:
- Chart.js 또는 Plotly 시각화 3개 이상
- 다크/라이트 모드 토글
- 영문 README.md 작성
- 커밋 5개 이상 (의미 있는 커밋 메시지)
- 반응형 디자인 (모바일 대응)
- 힌트 없음 — 스스로 설계하고 AI를 활용

**평가 기준**: 위 요구사항 모두 충족 + 디자인 완성도 + GitHub 커밋 히스토리

## 모범 답안

각 난이도의 예시 코드는 아래 폴더에서 확인:

```
examples/
├── 하/
│   └── index.html     ← 반도체 공정 엔지니어 지망생 예시
├── 중/
│   └── index.html     ← 2차전지 품질 엔지니어 지망생 예시
└── 상/
    ├── index.html     ← 디스플레이 공정 엔지니어 예시
    └── README.md      ← 영문 README 예시
```

> 모범 답안을 먼저 보는 것보다 스스로 시도한 후 비교하는 것을 권장합니다.

## 제출 방법

1. GitHub Pages URL을 강의 플랫폼 제출란에 입력
2. 사용한 Gemini 프롬프트 기록 첨부 (텍스트 파일 또는 스크린샷)
3. (상 도전자) GitHub 레포지토리 URL도 함께 제출

**제출 마감**: 5차시 시작 전

## 자주 묻는 질문

**Q. 코딩을 전혀 모르는데 할 수 있나요?**
A. 네, 가능합니다. '하' 레벨은 프롬프트 복붙과 파일 업로드만으로 완성할 수 있도록 설계되었습니다.

**Q. Gemini가 생성한 코드가 작동하지 않아요.**
A. "이 코드를 실행하면 [오류 내용] 이 나와. 고쳐줘" 라고 Gemini에게 오류 메시지를 그대로 붙여넣으세요.

**Q. 디자인을 바꾸고 싶어요.**
A. "배경색을 #1a1a2e로 바꾸고, 버튼에 호버 애니메이션 추가해줘" 처럼 구체적으로 요청하세요.

**Q. 내용을 영어로 바꿔도 되나요?**
A. '상' 레벨은 영문도 가능합니다. '하'/'중'은 한국어 유지 권장.
