카카오톡 대신 텔레그램인 이유

카카오톡 봇 vs 텔레그램 봇

카카오톡 봇의 현실

  • 카카오 비즈니스 계정 필수
  • 심사 및 승인 프로세스 수 주 소요
  • API 호출당 요금 발생
  • 메시지 형식 제약 심각
  • 한국에서만 유용

텔레그램 봇의 현실

  • @BotFather에서 /newbot 한 줄
  • 즉시 발급, 심사 없음
  • 완전 무료, 제한 없음
  • 파일, 이미지, 인라인 키보드 전부 OK
  • 전 세계 9억 명 사용자

핵심 인사이트

텔레그램 봇 API + Claude API = 내 스마트폰이 24시간 AI 비서로 변신. 디스플레이 공장 수율 알람도 이것으로 충분합니다.

텔레그램 봇 생태계 — 9억 명의 플랫폼

“텔레그램은 단순한 메신저가 아닙니다. 봇 플랫폼입니다.”

전 세계에서 실제로 쓰이는 봇들

  • 쇼핑 알람 봇 — 가격 떨어지면 즉시 알림
  • 번역 봇 — 실시간 텍스트 번역
  • 주식/코인 봇 — 가격 모니터링
  • AI 어시스턴트 봇 — GPT, Claude 연결
  • 디스플레이 수율 알람 봇 ← 오늘 만들 것

봇이 할 수 있는 것들

  • 텍스트/이미지/파일 송수신
  • 인라인 키보드 버튼
  • 그룹 채팅방 관리
  • 결제 처리 (텔레그램 Stars)
  • 외부 API 연동 ← 핵심

Polling vs Webhook — 어떤 방식을 쓸까?

Polling vs Webhook 비교 — 오늘은 Polling으로 시작합니다
항목 Polling Webhook
작동 방식 봇이 주기적으로 메시지 확인 텔레그램이 서버에 push
서버 필요? 불필요 (로컬 실행 가능) 필요 (HTTPS 서버)
속도 약간 느림 (~0.5초) 빠름 (즉시)
개발 난이도 쉬움 보통
추천 상황 개발/테스트/학습 프로덕션 배포

오늘 강의 전략

로컬에서 Polling 방식으로 빠르게 프로토타입을 만들고, 실제 배포 시에는 Webhook으로 전환하면 됩니다.

기본 봇 코드 — 뼈대 만들기

from telegram import Update
from telegram.ext import Application, CommandHandler, MessageHandler, filters
import os

# 환경 변수에서 토큰 로드
TOKEN = os.getenv("TELEGRAM_BOT_TOKEN")

# /start 명령어 핸들러
async def start(update: Update, context):
    await update.message.reply_text(
        "안녕하세요! 디스플레이 AI 어시스턴트입니다.\n\n"
        "OLED/LCD 공정, 수율 관리, 디스플레이 기술 전반에 대해\n"
        "전문적인 답변을 드릴 수 있습니다.\n\n"
        "/help - 사용 가능한 명령어 보기"
    )

# /help 명령어 핸들러
async def help_cmd(update: Update, context):
    await update.message.reply_text(
        "*명령어 목록*\n\n"
        "/yield - 수율 현황 조회\n"
        "/analyze - 데이터 분석 요청\n"
        "/report - 일일 보고서 생성\n"
        "/ask [질문] - AI에게 디스플레이 질문",
        parse_mode="Markdown"
    )

# 앱 구성 및 실행
app = Application.builder().token(TOKEN).build()
app.add_handler(CommandHandler("start", start))
app.add_handler(CommandHandler("help", help_cmd))
app.run_polling()

Claude API 연결 — AI 두뇌 이식

import anthropic
import os

# Claude 클라이언트 초기화
claude = anthropic.Anthropic(api_key=os.getenv("ANTHROPIC_API_KEY"))

# 디스플레이 전문가 시스템 프롬프트
SYSTEM_PROMPT = """당신은 삼성디스플레이/LG디스플레이 전문 AI 어시스턴트입니다.
OLED, LCD, 수율(yield) 관리, TFT 공정 기술, 색재현율,
휘도, 번인 현상 등 디스플레이 산업 전반에 대해 전문적으로 답변하세요.
답변은 간결하고 실용적으로, 핵심만 전달하세요.
모르는 것은 모른다고 정직하게 말하세요."""

# 사용자별 대화 히스토리 저장
conversation_history = {}

async def handle_message(update: Update, context):
    user_id = update.message.from_user.id
    user_message = update.message.text

    # 사용자별 대화 히스토리 유지 (멀티턴 대화)
    if user_id not in conversation_history:
        conversation_history[user_id] = []

    conversation_history[user_id].append(
        {"role": "user", "content": user_message}
    )

    # Claude API 호출
    response = claude.messages.create(
        model="claude-3-5-sonnet-20241022",
        max_tokens=500,
        system=SYSTEM_PROMPT,
        messages=conversation_history[user_id]
    )

    reply = response.content[0].text
    conversation_history[user_id].append(
        {"role": "assistant", "content": reply}
    )
    await update.message.reply_text(reply)

완성된 봇 코드 — 전체 통합

# display_bot.py — 완성본
import os, anthropic
from telegram import Update
from telegram.ext import Application, CommandHandler, MessageHandler, filters
from dotenv import load_dotenv

load_dotenv()
TOKEN = os.getenv("TELEGRAM_BOT_TOKEN")
claude = anthropic.Anthropic(api_key=os.getenv("ANTHROPIC_API_KEY"))

SYSTEM_PROMPT = """당신은 삼성디스플레이/LG디스플레이 전문 AI 어시스턴트입니다.
OLED, LCD, 수율(yield) 관리, TFT 공정 기술에 대해 전문적으로 답변하세요.
답변은 간결하게, 핵심만 전달하세요."""

async def start(update, context):
    await update.message.reply_text(
        "안녕하세요! 디스플레이 AI 어시스턴트입니다.\n"
        "OLED/LCD 공정, 수율 관리 질문을 해보세요!\n"
        "/help - 명령어 목록"
    )

async def yield_cmd(update, context):
    import random
    line = " ".join(context.args) if context.args else "전체"
    current = round(random.uniform(93, 99), 1)
    prev = round(current + random.uniform(-1, 1), 1)
    delta = round(current - prev, 1)
    status = "정상 범위" if current >= 95 else "임계값 이하! 즉시 확인 요망"
    msg = (
        f"*{line} 수율 현황*\n\n"
        f"현재 수율: `{current}%`\n"
        f"전시간 대비: `{delta:+.1f}%p`\n"
        f"{status}"
    )
    await update.message.reply_text(msg, parse_mode="Markdown")

async def handle_message(update, context):
    resp = claude.messages.create(
        model="claude-3-5-sonnet-20241022",
        max_tokens=400,
        system=SYSTEM_PROMPT,
        messages=[{"role": "user", "content": update.message.text}]
    )
    await update.message.reply_text(resp.content[0].text)

app = Application.builder().token(TOKEN).build()
app.add_handler(CommandHandler("start", start))
app.add_handler(CommandHandler("yield", yield_cmd))
app.add_handler(MessageHandler(
    filters.TEXT & ~filters.COMMAND, handle_message
))
app.run_polling()
