AI가 가장 빠르게 변하는 지금

6개월 전에 배운 것도 이미 구식이 됩니다.

중요한 건 “AI를 배웠다”가 아니라 “AI와 함께 계속 배울 수 있다”는 능력입니다.

그리고 그 능력을 — 오늘 여러분은 이미 가지고 있습니다.

이 강의에서 증명된 것

  • 16강 동안 코드를 두려워하지 않게 됨
  • AI API를 직접 연결해봄
  • 실제로 동작하는 앱을 배포해봄
  • 디스플레이 산업 맥락에서 AI를 적용해봄

오늘의 핵심

AI 트렌드를 읽는 눈 + 내 분야에 적용하는 능력. 이 두 가지가 미래의 경쟁력입니다.

2025 AI 지형도 변화 타임라인

GitHub Trending — 전 세계 개발자들이 지금 만드는 것

  • AI 에이전트
  • RAG
  • 멀티모달
  • 오픈소스 LLM

지금 가장 뜨거운 분야

  • AutoGPT / CrewAI — 목표를 주면 AI가 스스로 계획하고 실행
  • LangGraph — 복잡한 AI 워크플로우를 그래프로 표현
  • Claude Code (여러분이 배운 것!) — 코드베이스 전체를 이해하는 AI
  • 핵심 개념: 에이전트가 도구를 선택하고, 결과를 보고, 다음 행동을 결정

기업에서 가장 많이 쓰는 실용 AI

  • LlamaIndex — 문서를 AI가 읽을 수 있는 형태로 변환
  • Chroma / Weaviate — 벡터 데이터베이스 (의미 기반 검색)
  • 실제 사례: 삼성 제품 매뉴얼 Q&A 봇, 공정 표준 SOP 검색 시스템
  • RAG = Retrieval-Augmented Generation (검색 증강 생성)

텍스트를 넘어서

  • GPT-4o, Gemini 1.5 — 이미지 + 텍스트 동시 처리
  • 디스플레이 응용: 불량 이미지를 AI에게 보여주면 원인 분석
  • 음성 입력 → 텍스트 변환 → AI 답변 → 음성 출력 파이프라인
  • 현장 엔지니어가 사진 찍으면 즉시 AI 진단

무료로 로컬에서 실행 가능

  • DeepSeek R1 — 중국발 오픈소스, GPT-4급 성능
  • Llama 3.3 — Meta의 오픈소스, 기업 활용 가능
  • Mistral — 유럽발 소형 고성능 모델
  • 장점: API 비용 없음, 데이터 외부 전송 없음, 사내 보안 환경 적합

LG디스플레이 - OLED 증착 공정 자동 최적화 연구 - AI가 온도/압력/증착 속도를 실시간 조정 - 엔지니어의 경험을 데이터로 형식화

AI가 엔지니어를 대체하나요?

아니요.

AI는 데이터를 처리합니다. 엔지니어는 판단을 내립니다.

AI가 잘하는 것

  • 24시간 데이터 모니터링
  • 패턴 인식 및 이상 감지
  • 방대한 이력 데이터 분석

엔지니어가 여전히 필요한 것

  • 새로운 문제 정의
  • 물리적 원인 파악
  • 최종 의사결정 및 책임

AI 생태계 성장 트렌드

library(ggplot2)

growth <- data.frame(
  Year = rep(2020:2025, 3),
  Value = c(
    10,  15,  25,  45,  80,  150,   # AI 스타트업 수 (지수 기준)
     5,   8,  15,  30,  60,  120,   # AI 관련 채용 공고
     2,   3,   8,  20,  50,  110    # GitHub AI 레포 수
  ),
  Category = rep(
    c("AI 스타트업", "AI 채용공고", "GitHub AI 레포"),
    each = 6
  )
)

ggplot(growth, aes(x = Year, y = Value,
                   color = Category, group = Category)) +
  geom_line(linewidth = 1.5) +
  geom_point(size = 3) +
  scale_color_manual(values = c(
    "AI 스타트업"     = "#E74C3C",
    "AI 채용공고"     = "#3498DB",
    "GitHub AI 레포"  = "#27AE60"
  )) +
  scale_y_continuous(labels = function(x) paste0(x, "x")) +
  theme_minimal(base_size = 13) +
  labs(
    title    = "AI 생태계 성장 트렌드 (2020 = 10 기준)",
    subtitle = "2020~2025년 5년간 10~15배 성장 — 지금이 기회",
    x = "연도", y = "성장 지수", color = ""
  )
