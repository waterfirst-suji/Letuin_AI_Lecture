Streamlit vs Shiny 비교

  • 언어 및 구조

  • Streamlit: Python 기반
  • Shiny: R 기반
  • Streamlit: 빠른 프로토타입

Shiny ui 함수 정리

Shiny ui 함수 정리표
Function Description
fluidPage Creates a fluid page
sidebarLayout Layout with sidebar
mainPanel Main panel for output
actionButton Button for actions

auto-animate: Shiny 코드 변환

UI 코드

ui <- fluidPage(
  titlePanel("수율 트렌드 대시보드"),
  sidebarLayout(
    sidebarPanel(
      fileInput("file", "CSV 파일 업로드"),
      dateRangeInput("date", "날짜 범위")
    ),
    mainPanel(
      plotOutput("yieldPlot")
    )
  )
)
