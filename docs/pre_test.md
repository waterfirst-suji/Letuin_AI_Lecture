## [A] 기본 컴퓨터 & 인터넷 개념 (1~5번)

**1.** 파일 확장자와 용도가 올바르게 연결된 것은?
- ① `.exe` — 이미지 파일
- ② `.csv` — 쉼표로 구분된 표 데이터 ✅
- ③ `.mp4` — 텍스트 문서
- ④ `.pdf` — 실행 파일

**2.** API(Application Programming Interface)의 역할로 가장 적절한 것은?
- ① 컴퓨터 비밀번호 관리 시스템
- ② 소프트웨어 간 데이터를 주고받는 통신 규약 ✅
- ③ 파일을 압축하는 방법
- ④ 화면을 디자인하는 도구

**3.** 터미널에서 현재 디렉토리의 파일 목록을 보는 명령어는?
- ① `cd`
- ② `ls` (Linux/Mac) 또는 `dir` (Windows) ✅
- ③ `rm`
- ④ `mkdir`

**4.** "환경변수(Environment Variable)"의 올바른 설명은?
- ① 프로그램 실행 중 변하는 화면 변수
- ② 운영체제 수준에서 설정되는 구성 값 (API Key 등 저장에 사용) ✅
- ③ 데이터베이스 내 변수
- ④ HTML 태그 내 속성 값

**5.** 오픈소스(Open Source) 소프트웨어의 특징으로 옳은 것은?
- ① 반드시 무료로만 사용 가능
- ② 소스 코드가 공개되어 누구나 사용/수정/배포 가능 ✅
- ③ 정부가 관리하는 공공 소프트웨어
- ④ 기업이 만들면 오픈소스가 될 수 없음

## [B] 프로그래밍 기초 개념 (6~11번)

**6.** 다음 Python 코드의 출력 결과는?
```python
data = [10, 20, 30, 40, 50]
print(data[2])
```
- ① 20
- ② 30 ✅
- ③ 40
- ④ 오류 발생

**7.** 함수(Function)에 대한 설명으로 옳은 것은?
- ① 반드시 숫자를 반환해야 하는 코드
- ② 특정 작업을 수행하는 재사용 가능한 코드 블록 ✅
- ③ 한 번만 실행되는 코드 묶음
- ④ 수학의 방정식과 완전히 동일한 개념

**8.** 다음 중 패키지 관리자(Package Manager)가 아닌 것은?
- ① pip (Python)
- ② npm (Node.js)
- ③ conda (Anaconda)
- ④ GitHub ✅ (코드 저장소)

**9.** `if-else` 조건문이 필요한 상황으로 가장 적절한 것은?
- ① 100개의 데이터를 순서대로 처리할 때
- ② 수율이 95% 이상이면 "양품", 미만이면 "불량"으로 분류할 때 ✅
- ③ 두 숫자를 더할 때
- ④ 함수를 정의할 때

**10.** 다음 중 데이터 타입이 나머지 셋과 다른 하나는?
- ① `42`
- ② `3.14`
- ③ `"hello"` ✅ (문자열, 나머지는 숫자형)
- ④ `-7`

**11.** JSON 형식에 대한 설명으로 옳은 것은?
- ① 이미지 압축 형식
- ② 키-값 쌍으로 구성된 경량 데이터 교환 형식 ✅
- ③ 동영상 인코딩 방식
- ④ 데이터베이스 쿼리 언어

## [C] 데이터 & 통계 기초 (12~16번)

**12.** 평균(Mean)과 중앙값(Median)에 대한 설명으로 옳은 것은?
- ① 평균은 항상 중앙값보다 크다
- ② 이상값(outlier)이 있을 때 중앙값이 더 안정적이다 ✅
- ③ 두 값은 항상 동일하다
- ④ 중앙값은 모든 값을 더해 개수로 나눈 값이다

**13.** 상관관계(Correlation)에 대한 설명으로 **옳지 않은** 것은?
- ① 두 변수 사이의 관계 강도와 방향을 나타낸다
- ② 상관계수는 -1에서 1 사이의 값을 가진다
- ③ 상관관계가 높으면 반드시 인과관계가 있다 ✅ ← 틀린 설명
- ④ 0에 가까울수록 관계가 약하다

**14.** 디스플레이 산업에서 "수율(Yield)"의 정의는?
- ① 생산된 전체 패널 수
- ② 전체 생산량 대비 양품 비율 ✅
- ③ 패널의 최대 밝기
- ④ 공정 처리 속도

**15.** 파이 차트(Pie Chart)의 가장 적절한 사용 용도는?
- ① 시간에 따른 트렌드 확인
- ② 두 변수 간의 관계 확인
- ③ 전체 중 각 항목의 비율 표시 ✅
- ④ 데이터 분포와 이상값 탐지

**16.** 표준편차(Standard Deviation)에 대한 설명으로 옳은 것은?
- ① 데이터의 가운데 값
- ② 데이터가 평균으로부터 얼마나 퍼져 있는지를 나타내는 지표 ✅
- ③ 데이터의 최대값과 최소값의 차이
- ④ 데이터 개수를 나타내는 값

## [D] AI & 최신 기술 개념 (17~22번)

**17.** LLM(Large Language Model)에 대한 설명으로 옳은 것은?
- ① 이미지를 분류하는 AI 모델
- ② 대규모 텍스트 데이터로 사전 학습된 언어 처리 AI ✅
- ③ 로봇 제어에 특화된 AI
- ④ 수치 예측만 가능한 회귀 모델

**18.** "바이브 코딩(Vibe Coding)"을 처음 제안한 AI 연구자는?
- ① Sam Altman
- ② Andrej Karpathy ✅
- ③ Geoffrey Hinton
- ④ Yann LeCun

**19.** AI 에이전트(Agent)의 특징으로 가장 적절한 것은?
- ① 사람이 모든 단계를 직접 지시해야 한다
- ② 목표를 주면 스스로 계획하고 도구를 사용해 실행한다 ✅
- ③ 텍스트만 생성할 수 있다
- ④ 인터넷 연결 없이 작동한다

**20.** 프롬프트 엔지니어링에서 "System Prompt"의 역할은?
- ① 사용자의 첫 번째 질문
- ② AI의 역할과 행동 방식을 사전에 정의하는 지시 ✅
- ③ AI의 학습 데이터 목록
- ④ AI 응답 속도를 높이는 설정

**21.** MCP(Model Context Protocol)에 대한 설명으로 옳은 것은?
- ① 메모리 캐시 프로토콜
- ② AI 모델이 외부 도구/데이터와 표준화된 방식으로 연동하는 프로토콜 ✅
- ③ 모바일 통신 프로토콜
- ④ 멀티코어 처리 방식

**22.** RAG(Retrieval-Augmented Generation)이란?
- ① AI가 랜덤하게 답변을 생성하는 방식
- ② 외부 데이터베이스를 검색하여 관련 정보를 가져온 후 답변을 생성하는 방식 ✅
- ③ 이미지를 텍스트로 변환하는 기술
- ④ 음성 인식 후 답변 생성 방식

## [F] 디스플레이 기초 공정 지식 (27~35번)

**27.** OLED와 LCD의 발광 방식 차이로 옳은 것은?
- ① OLED는 백라이트가 필요, LCD는 자체 발광
- ② OLED는 자체 발광, LCD는 백라이트가 필요 ✅
- ③ 두 기술 모두 자체 발광 방식
- ④ 두 기술 모두 백라이트가 필요

**28.** TFT(Thin Film Transistor)의 역할로 옳은 것은?
- ① 디스플레이의 색상을 결정하는 필터
- ② 각 픽셀의 전류를 제어하는 스위치 소자 ✅
- ③ 빛의 밝기를 균일하게 분산하는 층
- ④ 패널 봉지(Encapsulation)를 담당하는 층

**29.** 포토리소그래피(Photolithography) 공정의 목적으로 옳은 것은?
- ① 유리 기판을 절단하는 공정
- ② 빛(UV)을 이용해 회로 패턴을 기판에 전사하는 공정 ✅
- ③ 박막에 가스를 주입하는 공정
- ④ 봉지재를 도포하는 공정

**30.** 포토리소그래피 공정 순서로 올바른 것은?
- ① 노광 → 현상 → 포토레지스트 도포 → 식각
- ② 포토레지스트 도포 → 노광 → 현상 → 식각 ✅
- ③ 식각 → 노광 → 현상 → 포토레지스트 도포
- ④ 현상 → 포토레지스트 도포 → 노광 → 식각

**31.** 드라이 에치(Dry Etch) 공정의 특징으로 옳은 것은?
- ① 물이나 액체 화학물질을 사용하는 습식 식각 방법
- ② 플라즈마나 반응성 가스를 이용해 박막을 제거하는 방식 ✅
- ③ 레이저를 사용해 기판을 절단하는 방법
- ④ 열을 이용해 박막을 증착하는 방법

**32.** 드라이 에치에서 "이방성 식각(Anisotropic Etching)"의 의미는?
- ① 모든 방향으로 균일하게 식각
- ② 수직 방향으로 우선적으로 식각되어 선명한 패턴을 형성 ✅
- ③ 수평 방향으로만 식각
- ④ 가장 빠른 속도로 식각

**33.** 잉크젯 공정(Inkjet Process)이 OLED 제조에 활용될 때의 장점으로 옳은 것은?
- ① 진공 환경에서만 공정 가능
- ② 유기 발광 재료를 정확한 위치에 도포하여 재료 낭비 최소화 ✅
- ③ 기존 증착 대비 해상도가 낮음
- ④ 대면적 패널에는 적용 불가

**34.** 잉크젯 프린팅 공정에서 "커피링 효과(Coffee Ring Effect)"가 문제가 되는 이유는?
- ① 잉크가 너무 빠르게 건조되기 때문
- ② 건조 과정에서 잉크 성분이 불균일하게 분포되어 박막 두께 균일성이 저하되기 때문 ✅
- ③ 잉크 점도가 너무 낮기 때문
- ④ 기판 온도가 너무 높기 때문

**35.** 디스플레이 색 재현율 표준 중 영화/전문 영상에 주로 사용되는 것은?
- ① sRGB
- ② DCI-P3 ✅
- ③ YCbCr
- ④ NTSC
