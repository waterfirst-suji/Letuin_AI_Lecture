자소서 작성 전략

STAR 기법 + 프로젝트 증명

❌ 나쁜 예: > “Python과 AI를 활용한 데이터 분석 능력을 보유하고 있습니다.”

→ 추상적이고 증명 불가

✅ 좋은 예 (STAR + 증거):

S (Situation): 디스플레이 제조 공정에서 수율 저하 문제를 발견했습니다.

T (Task): 실시간 모니터링 및 이상 탐지 시스템 구축을 목표로 했습니다.

A (Action): Python Streamlit과 Claude AI를 활용해 자동 감지 시스템을 개발했습니다. 텔레그램 API로 즉시 알림을 보내도록 구현했습니다.

R (Result): 3주 만에 MVP를 완성하고 Streamlit Cloud에 배포했습니다. GitHub에 전체 코드를 공개하고 문서화했습니다.

증거: [GitHub 링크] | [라이브 데모]
