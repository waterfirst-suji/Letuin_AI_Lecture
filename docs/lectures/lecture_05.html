실습 앱 — 디스플레이 PPI 계산기

실제로 작동하는 앱을 만들어봅니다. PPI = Pixels Per Inch = 화면 선명도의 핵심 지표.

import streamlit as st
import math

st.title("📱 디스플레이 PPI 계산기")
st.caption("Pixels Per Inch — 화면 선명도의 핵심 지표")

col1, col2, col3 = st.columns(3)
with col1:
    width_px = st.number_input("가로 픽셀", value=2560, step=1)
with col2:
    height_px = st.number_input("세로 픽셀", value=1600, step=1)
with col3:
    diagonal_inch = st.number_input("화면 대각선 (인치)", value=14.0, step=0.1)

if st.button("PPI 계산"):
    ppi = math.sqrt(width_px**2 + height_px**2) / diagonal_inch
    st.metric("PPI", f"{ppi:.1f}")
    if ppi >= 300:
        st.success("Retina 수준 — 사람 눈으로 픽셀 구분 불가")
    elif ppi >= 200:
        st.info("고해상도 — 일반 업무에 충분")
    else:
        st.warning("저해상도 — 텍스트 가독성 떨어질 수 있음")

Note

Apple이 iPhone 4에서 처음 “Retina Display” 기준을 300 PPI로 설정. 삼성 Galaxy S24는 416 PPI로 이 기준을 크게 웃돕니다.

2단계: GitHub에 Push

git init
git add .
git commit -m "feat: PPI 계산기 앱 추가"
git remote add origin https://github.com/YOUR_ID/ppi-calc.git
git push -u origin main

3단계: Streamlit Cloud 배포

  1. share.streamlit.io 접속
  2. GitHub 계정으로 로그인
  3. “New app” 클릭
  4. Repository / Branch / 파일 선택
  5. “Deploy!” 클릭
https://YOUR-APP-NAME.streamlit.app

완료. 전 세계 누구나 접속 가능.

import streamlit as st

# 반드시 첫 줄에 위치 — 탭 제목, 아이콘, 레이아웃 설정
st.set_page_config(
    page_title="디스플레이 분석 도구",
    page_icon="🖥️",
    layout="wide",                    # 넓은 레이아웃 (강력 추천)
    initial_sidebar_state="expanded"
)

# 로딩 스피너
with st.spinner("데이터 분석 중..."):
    result = run_analysis()

st.success("✅ 분석 완료!")

Note

layout="wide" 설정 하나로 앱이 훨씬 전문적으로 보입니다. 디스플레이 분석 앱처럼 차트가 많은 경우 필수입니다.

자주 하는 실수와 올바른 패턴

피해야 할 패턴

# ❌ 매번 재실행 — 느림
def load_big_data():
    df = pd.read_csv("huge_file.csv")
    return df

# ❌ session_state 없이 상태 관리
count = 0
if st.button("클릭"):
    count += 1  # 항상 1로 리셋

올바른 패턴

# ✅ 캐싱으로 한 번만 실행
@st.cache_data
def load_big_data():
    df = pd.read_csv("huge_file.csv")
    return df

# ✅ session_state로 상태 유지
if "count" not in st.session_state:
    st.session_state.count = 0
if st.button("클릭"):
    st.session_state.count += 1

핵심 원리: Streamlit은 인터랙션이 발생할 때마다 전체 스크립트를 처음부터 다시 실행합니다. @st.cache_data와 st.session_state는 이를 효율적으로 처리하는 도구입니다.
