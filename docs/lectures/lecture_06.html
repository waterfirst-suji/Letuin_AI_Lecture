Shiny의 세 가지 핵심 개념

1. ui — 레이아웃

사용자가 보는 화면. 버튼, 슬라이더, 차트 위치를 정의.

ui <- fluidPage(
  titlePanel("수율 대시보드"),
  sidebarLayout(
    sidebarPanel(selectInput("line", "라인", choices=c("A","B","C"))),
    mainPanel(plotOutput("yieldPlot"))
  )
)

2. server — 로직

데이터를 처리하는 엔진. input$으로 사용자 입력을 받고, output$으로 결과를 내보냄.

server <- function(input, output) {
  output$yieldPlot <- renderPlot({ ... })
}

3. reactive — 마법 ✨

입력이 바뀌면 출력이 자동으로 바뀐다. 새로고침 불필요.

filtered_data <- reactive({ df %>% filter(Line == input$line) })

수율 트렌드 차트 (실제 데이터 시뮬레이션)

라인별 평균 수율 비교 (±1σ 오차 막대)

Shiny server 로직 완성하기

library(shiny)
library(ggplot2)
library(dplyr)

server <- function(input, output) {

  # reactive: 입력값이 바뀌면 자동 재계산
  filtered_data <- reactive({
    data <- yield_df
    if (input$line != "전체") {
      data <- data %>% filter(Line == input$line)
    }
    data %>%
      filter(Date >= input$dateRange[1],
             Date <= input$dateRange[2])
  })

  # renderPlot: filtered_data()가 바뀌면 차트도 자동 업데이트
  output$yieldPlot <- renderPlot({
    ggplot(filtered_data(), aes(x = Date, y = Yield, color = Line)) +
      geom_line(linewidth = 0.9) +
      geom_hline(yintercept = 95, linetype = "dashed", color = "#E74C3C") +
      theme_minimal(base_size = 13) +
      labs(title = "수율 트렌드", y = "수율 (%)")
  })

  # renderTable: 요약 통계표
  output$summaryTable <- renderTable({
    filtered_data() %>%
      group_by(Line) %>%
      summarise(평균수율 = mean(Yield), 최솟값 = min(Yield),
                최댓값 = max(Yield), .groups = "drop")
  })
}

shinyapps.io 배포 — 4단계

1단계: rsconnect 설치 및 인증

install.packages("rsconnect")
library(rsconnect)

rsconnect::setAccountInfo(
  name   = "YOUR_ACCOUNT_NAME",
  token  = "YOUR_TOKEN",
  secret = "YOUR_SECRET"
)

2단계: 앱 배포

# 앱 폴더 전체 배포
rsconnect::deployApp("./myapp")

# 또는 현재 디렉토리
rsconnect::deployApp()

3단계: Token 발급 위치

  1. shinyapps.io 로그인
  2. 우측 상단 프로필 → Tokens
  3. Show Secret 클릭 후 복사

4단계: 배포 확인

https://YOUR_ID.shinyapps.io/myapp/

Tip

무료 플랜 제한 - 월 25시간 활성 시간 - 5개 앱까지 무료 - 포트폴리오용으로는 충분합니다

shinydashboard — 더 전문적인 레이아웃

library(shiny)
library(shinydashboard)

ui <- dashboardPage(
  dashboardHeader(title = "수율 모니터링"),

  dashboardSidebar(
    sidebarMenu(
      menuItem("트렌드", tabName = "trend", icon = icon("chart-line")),
      menuItem("라인 비교", tabName = "compare", icon = icon("bar-chart")),
      menuItem("불량 분석", tabName = "defect", icon = icon("exclamation-triangle"))
    )
  ),

  dashboardBody(
    tabItems(
      tabItem(tabName = "trend",
        fluidRow(
          valueBox(96.5, "평균 수율", icon = icon("percent"), color = "blue"),
          valueBox(3.5,  "불량률",   icon = icon("times"),   color = "red"),
          valueBox(12540, "일 생산량", icon = icon("tv"),    color = "green")
        ),
        fluidRow(box(plotOutput("trendPlot"), width = 12))
      )
    )
  )
)
